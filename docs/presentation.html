문제 정의

당근마켓 "인테리어 배우기"
에서 매일 반복되는 일

"도배 비용이 얼마에요?" — 같은 질문이 하루에도 3번씩
"인터넷에 다 나와 있는데 왜 여기서 물어봐요" — 짜증난 답변자
"견적 받았는데 바가지인지 어떻게 알아요?" — 불안한 집주인

정보는 인터넷에 있지만 모아져 있지 않고, 신뢰할 수 있는지도 알 수 없는 상황

정보 축적

핵심 기능

v0.1 구현된 기능

  • 30개+ Q&A 지식베이스 (10개 카테고리)
  • 키워드 검색 + 결과 하이라이팅
  • 최근 검색어 + 태그 자동완성
  • 오늘의 팁 (날짜 기반 추천)
  • 관련 유튜브·블로그 링크
  • 공사 비용 간단 계산기
  • 즐겨찾기 + 내 질문 추가
  • 데이터 JSON 내보내기/가져오기
  • 🥇🥈🥉 인기 순위 뱃지
  • 공유하기 (Web Share API)
