# PRD — 집수리 가이드

**Product Requirements Document**
버전: 0.1.0 | 작성일: 2026-04-22

---

## 1. 배경 및 문제 정의

### 1.1 문제

당근마켓 "인테리어 배우기" 커뮤니티에서 반복적으로 관찰되는 패턴:

- 같은 질문이 반복 게시됨 (도배 비용, 화장실 수리비 등)
- 질문이 모호해 답변자가 짜증을 내는 상황 발생
- 인터넷에 이미 정보가 있지만 찾기 어렵거나 신뢰할 수 없음
- 비용 정보 특히 부정확하고 분산되어 있음

### 1.2 기회

집수리·인테리어 정보는:
- 일반적이고 반복적인 패턴이 있음 (비용, 공정 순서, 주의사항)
- 모아두면 커뮤니티 전체에 가치를 제공할 수 있음
- 사용자의 경험담이 축적될수록 품질이 향상됨

---

## 2. 제품 비전

> "집수리 정보를 검색하면 바로 나오고, 관련 영상·블로그도 함께 알려주는 모바일 가이드"

---

## 3. 사용자 페르소나

### 메인 페르소나: 집수리 초보 (30-50대)
- 생애 첫 집수리를 앞두고 있거나 진행 중
- 인터넷 정보는 넘치지만 신뢰할 수 있는 정리된 정보가 없어 불안
- 가격이 적정한지 모르고 업체에 속을까 걱정
- 모바일 위주 사용

### 서브 페르소나: 커뮤니티 정보 제공자
- 집수리 경험이 풍부하고 정보를 나누고 싶음
- 같은 질문에 반복 답변하는 것이 피곤함

## 5. 기능 요구사항

### 5.1 검색 (MVP)
- 키워드 검색 (질문·답변·태그·카테고리 통합)
- 검색 결과 하이라이팅
- 최근 검색어 기록 (최대 8개)
- 태그 자동완성 제안
- 관련 없으면 "질문 추가하기" 유도

### 5.2 지식베이스
- 30개+ Q&A 사전 등재
- 카테고리: 도배, 바닥재, 화장실, 샤시창호, 주방, 전기조명, 방수누수, 도장페인트, 철거구조, 기타
- 각 항목: 질문, 답변, 태그, 유튜브/블로그 링크
- 인기 순위 (좋아요 기반)

### 5.3 사용자 참여
- 질문 추가 (카테고리, 질문, 답변, 태그, 참고 링크)
- 좋아요 누르기
- 즐겨찾기 저장/해제
- 태그 클릭으로 관련 검색

### 5.4 데이터 관리
- localStorage 영속성
- JSON 내보내기/가져오기 (백업/복원)

## 6. 비기능 요구사항

| 항목 | 목표 |
|------|------|
| 성능 | 초기 로드 2초 이내 (gzip 80KB 이하) |
| 오프라인 | PWA manifest (홈화면 추가) |
| 접근성 | aria-label, role 기본 적용 |
| 모바일 | max-width 430px, 하단 탭 네비게이션 |
| 서버 | 없음 (클라이언트 전용) |

## 7. 현재 한계 (향후 개선사항)

| 기능 | 이유 | 필요 사항 |
|------|------|----------|
| AI 질문 답변 | API 비용, 서버 없음 | Claude API + 서버 |
| 커뮤니티 공유 | 서버 없음 | Backend + DB |
| 자동 정보 수집 | 크롤링 서버 없음 | 크롤러 + 스케줄러 |
| 지역 업체 정보 | 데이터 없음 | 외부 API + 데이터 |
| 사용자 계정 | 서버 없음 | 인증 서비스 |
