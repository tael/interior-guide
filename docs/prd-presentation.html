정보는 인터넷에 있지만 모아져 있지 않고, 신뢰할 수 있는지도 알 수 없다. 커뮤니티에서 경험담은 쌓이지만 사라진다.

💡 기회 & 비전

집수리 정보는 패턴이 있다

📐

일반화 가능

도배·화장실·샤시 비용은 지역·면적으로 범위화 가능. 한 번 정리하면 재사용 가능.

🌱

축적될수록 강해짐

사용자 경험담 하나하나가 지식베이스가 된다. 커뮤니티가 커질수록 품질 향상.

📱

모바일 순간 검색

업체 상담 중에 바로 꺼내 확인. 설치 없이, 무료로.

"집수리 정보를 검색하면 바로 나오고,
관련 영상·블로그도 함께 알려주는 모바일 가이드"
👤 사용자 페르소나

누구를 위한 제품인가

🏠
메인 · 80%

집수리 초보 (30-50대)

  • 생애 첫 집수리를 앞두고 있거나 진행 중
  • 넘치는 인터넷 정보, 믿을 수 있는 것이 없어 불안
  • 가격이 적정한지 모르고 속을까 걱정
  • 모바일로 검색하며 업체와 협의
핵심 니즈: "지금 받은 견적이 적정한지 5초 안에 알고 싶다"
💬
서브 · 20%

커뮤니티 정보 제공자

  • 집수리 경험이 풍부하고 정보를 나누고 싶음
  • 같은 질문에 반복 답변하는 것이 피곤함
  • 한 번 정리해두면 링크만 공유하고 싶음
핵심 니즈: "내 경험을 한 번 올려두면 알아서 퍼지면 좋겠다"

⚙️ 기능 요구사항

v0.1에 구현된 핵심 기능

5.1 검색
  • 질문·답변·태그·카테고리 통합 검색
  • 검색어 하이라이팅
  • 최근 검색어 기록 (최대 8개)
  • 태그 자동완성 제안
  • 결과 없으면 질문 추가 유도
5.3 사용자 참여
  • 질문 직접 추가 (카테고리·답변·태그·링크)
  • 좋아요 / 즐겨찾기
  • 태그 클릭 관련 검색
5.2 지식베이스
  • 51개 Q&A (10개 카테고리)
  • 각 항목: 질문·답변·태그·링크·좋아요
  • 인기 순위 뱃지 (🥇🥈🥉)
  • 오늘의 팁 (날짜 기반 추천)
  • 관련 질문 섹션 (같은 카테고리)
5.4 데이터 관리
  • localStorage 자동 저장
  • JSON 내보내기 / 가져오기
  • 공사 비용 계산기
