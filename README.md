# 집수리 가이드 🏠

집수리·인테리어 궁금증을 해결하는 모바일 Q&A 앱

**지금 사용하기 → https://tael.github.io/interior-guide/**

설치 없이 모바일 브라우저에서 바로 실행. 홈화면에 추가하면 앱처럼 사용 가능.

## 주요 기능

### 정보 검색
- 도배·바닥재·화장실·샤시·주방·전기·방수·철거 등 30개 이상 Q&A
- 키워드 검색 + 결과 하이라이팅
- 태그 클릭으로 관련 정보 탐색
- 최근 검색어 기록 (최대 8개)
- 검색어 입력 시 태그 자동완성

### 카테고리 탐색
- 10개 카테고리별 모아보기
- 인기순/기본순 정렬
- 공사 비용 간단 계산기 (면적 × 단가)

### 내 정보 관리
- ⭐ 즐겨찾기 저장
- ✏️ 내 질문 추가 (로컬 저장)
- 참고 링크 URL 첨부
- 데이터 JSON 내보내기/가져오기

### UX
- 오늘의 팁 (날짜 기반 Q&A 추천)
- 🥇🥈🥉 인기 순위 뱃지
- Q&A 공유하기 (Web Share API + 클립보드)
- 스크롤 맨위로 버튼
- 탭 카운트 뱃지

## 기술 스택

- React 19 + TypeScript + Vite + Tailwind CSS v4
- Zustand (상태 관리)
- localStorage (데이터 영속성)
- 정적 사이트 — 서버 없음

## 실행

```bash
npm install
npm run dev
```

## 빌드

```bash
npm run build   # dist/에 정적 파일 생성
npm run preview # 로컬 프리뷰
```

## 현재 한계 및 향후 개선 예정

| 기능 | 현재 | 개선 방향 |
|------|------|-----------|
| AI 답변 | 정적 Q&A | Claude API 연동 |
| 커뮤니티 공유 | 로컬만 | 서버 + DB |
| 자동 정보 수집 | 수동 입력 | 크롤링 자동화 |
| 지역 업체 정보 | 미지원 | 지역 기반 지도 |
| 시공 사진 | 미지원 | 이미지 첨부 |
